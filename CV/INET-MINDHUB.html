INET-MINDHUB

Curso semi presencial, consistente en simular un entorno laboral real, utilizando el método de aula invertida (proceso de aprendizaje autodidacta a partir de instrucciones concretas de trabajo guiadas por un mentor) y stand up meeting diarias.

Tecnologías aprendidas: HTML, CSS, Bootstrap, Javascript puro, VUE, JQuery, JSON

Semi-face-to-face course, consisting of simulating a real work environment, using the flipped classroom method (self-taught learning process based on specific work instructions guided by a mentor) and daily stand-up meetings.

Acquired technologies: HTML, CSS, Bootstrap, Javascript, VUE, JQuery, JSON
